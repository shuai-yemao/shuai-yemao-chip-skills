## 🚀 快速开始

### 安装

```bash
# 克隆技能仓库
git clone https://github.com/shuai-yemao/shuai-yemao-chip-skills.git

# 复制技能到 Claude Code 目录
cp -r shuai-yemao-chip-skills/skills/* ~/.claude/skills/

# 清理
rm -rf shuai-yemao-chip-skills
```

### 使用

```bash
# 启动 Claude Code
claude

# 直接引用技能
/brainstorm my idea          # 使用 brainstorming 技能
/code-review my changes       # 使用 code-review 技能
/embedded MCU problem         # 使用 embedded 技能
/writing-plans new feature    # 使用 writing-plans 技能
```

## 💡 Skills vs Workflows

| 类型 | 用途 | 示例 |
|------|------|------|
| **Skills（本仓库）** | 具体工具和指南 | `build-cmake`, `i2c-bus`, `debug-gdb-openocd` |
| **Workflows** | 完整流程编排 | `firmware-development`, `testing-cicd` |

**使用建议**：
- 只需要单一功能 → 使用 Skills
- 需要完整开发流程 → 使用 Workflows
- Workflows 内部会调用 Skills 完成具体任务
