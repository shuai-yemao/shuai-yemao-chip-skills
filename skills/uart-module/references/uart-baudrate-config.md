# 波特率计算表与误差分析

## USARTDIV 公式回顾

```
OVER16 模式: USARTDIV = fCK / (16 × BaudRate)
OVER8  模式: USARTDIV = fCK / (8  × BaudRate)
```

BRR 寄存器编码：

```
OVER16: BRR = (mantissa << 4) | fraction         // fraction 4bit [0~15]
OVER8:  BRR = (mantissa << 4) | (fraction << 1)  // fraction 3bit [0~7]，bit0 强制 0
```

## OVER8 模式高速速查

OVER8 在 fCK 相同时使波特率上限翻倍：

| fCK | 最高 OVER16 | 最高 OVER8 |
|-----|-----------|-----------|
| 72 MHz | 4.5 Mbps | 9 Mbps |
| 84 MHz | 5.25 Mbps | 10.5 Mbps |
| 100 MHz | 6.25 Mbps | 12.5 Mbps |
| 120 MHz | 7.5 Mbps | 15 Mbps |

当需要的波特率 > fCK/16 时，必须切到 OVER8。代价：由于 fraction 精度降低（3bit vs 4bit），误差略大于 OVER16 模式。
